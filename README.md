### JAVA Escape Character:
**\t: Insert a tab in the text at this point.**

input: 
```
public class class082222 {
    public static void main(String[] args){
        System.out.println("china\tjapan\tlebron");
    }
}
```
output: 
```
china	japan	lebron
```
